kubectl add — examples

Manifests served for kubectl add, one directory per format. Apply any of them server-side:

kubectl add https://scaffoldly.github.io/kubectl-add/yaml/nginx.yaml

Available

  • yaml/nginx.yaml — nginx Deployment + Service
  • yaml/configmap.yaml — a demo ConfigMap
  • yaml/secret.yaml — a demo Secret
  • kustomization/kustomization.yaml — a kustomization pulling nginx + a Secret over HTTP and a local ConfigMap
  • helm/Chart.yaml — a minimal helm chart (rendered + applied)

Remove any of them with --remove:

kubectl add --remove https://scaffoldly.github.io/kubectl-add/yaml/nginx.yaml
